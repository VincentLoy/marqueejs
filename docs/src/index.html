Why Marquee JS

Modern, lightweight and flexible marquee solution for your web applications

Lightning Fast

Zero dependencies, optimized animations, and minimal DOM operations for buttery-smooth performance.

Fully Customizable

Multiple directions, dynamic content updates, custom speeds, and flexible styling options.

Modern API

TypeScript support, promise-based API, and modern browser features for better developer experience.

Examples

Basic Example

Basic marquee example 
marqueejs('.marquee-basic');

RTL mode

>>>   This text moves from left to right 
marqueejs('.marquee-rtl', {
                            direction: 'right',
                            speed: 150
                        });

With Separator

INFO Separated content 

                            
marqueejs('.marquee-separator', {
    separator: '•',
    gap: 40
});

Usage

Import and initialize MarqueeJS:

import { marqueejs } from 'marqueejs';

        marqueejs('.selector', {
            speed: 100,
            direction: 'left',
            pauseOnHover: true
        });
