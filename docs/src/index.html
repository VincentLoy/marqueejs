MarqueeJS

A modern, lightweight marquee plugin for the web

Basic Example

Basic marquee example 
marqueejs('.marquee-basic');

Left to right

>>>   This text moves from left to right 
marqueejs('.marquee-rtl', {
                direction: 'right',
                speed: 150
            });

With Separator

INFO Separated content 

marqueejs('.marquee-separator', {
  separator: '•',
  gap: 40
});

Fast with Pause on Hover

This moves fast but pauses on hover 
marqueejs('.marquee-fast', {
  speed: 200,
  pauseOnHover: true
});

Installation

npm install marqueejs

Usage

Import and initialize MarqueeJS:

import { marqueejs } from 'marqueejs';

marqueejs('.selector', {
    speed: 100,
    direction: 'left',
    pauseOnHover: true
});
